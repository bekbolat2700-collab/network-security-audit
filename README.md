# Network Security Audit

## Description
Performed a basic network security audit using Nmap and Wireshark to identify open ports, services, and potential vulnerabilities.

## Tools Used
- Nmap (Network Mapper)
- Wireshark (Network traffic analyzer)

## Steps Taken
1. Scanned local network with Nmap: nmap -sV 192.168.1.0/24
2. Identified open ports and running services
3. Captured traffic with Wireshark
4. Analyzed network packets for suspicious activity

## Findings
- Open ports detected: 22 (SSH), 80 (HTTP)
- Unused services identified
- Basic risk assessment completed
